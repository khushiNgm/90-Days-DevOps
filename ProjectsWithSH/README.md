## ⚡ How to Use
1. Make the script executable:
   chmod +x Project1.sh
2. Run the script:
   ./Project1.sh
   
## Troubleshooting
- Docker permission denied: run `newgrp docker` or log out/in.
- Port 80 conflict: stop Apache using `sudo systemctl stop apache2`.##
